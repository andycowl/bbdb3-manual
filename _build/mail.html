Mail¶

Apparently there are other mail readers for Emacs besides Gnus. Most of these mail handling programs are supported by BBDB.

As mailing handling is abstracted to a generic Mail user Agent (MUA), adding support for a new Emacs mail reader program should be relatively straightforward. [Hint: Look at the code for the existing readers].
